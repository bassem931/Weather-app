Weather App
Bringing the weather news to you

This is the website were you can conveniently find the weather of any city you can think of. The process is very simple and will be explained down below
all you have to do is know the city code of the city you want to know the weather in. The results will be returned which are the temperature and date

To use this website follow this instrucions:
  1. choose the city you want to know the weather in
  2. type the city-code in the text space available
  3. click the button after the text space and wait for the magic to happen :)










/
*(MM/DD/YYYY), All dates are in UTC time

Openweather API was used for this project.This api is used to send request in the form of city code and recieve several weather information as a result.This include temparture humidity wind speed as well as the location of the city code entered and date. This is all provided by Openweather Api the information is retrieved and displayed to the user.
